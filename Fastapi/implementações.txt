Projeto ApiGIBS já implementado com ambiente virtual venv, para caso de test da equipe,
já vai com as bibliotecas e ferramentas instaladas no ambiente virtual venv
para execução fastapi com request da API GIBS DA NASA!

Todas instalações de ferramentas e bibliotecas foram feitas neste ambiente incluindo a API.
Foi criada a app com uma lista data_fake que foram adcionados dados fakes semelhantes ao da Api Gibs da nasa.
criado 2 métodos gets, 1° com endereço =  ("/"),  para verificar se api estar funcionando com mensagem apresentada.
2° com endereço =  ("/Gibs/gibs_api"), para pegar os dados fakes criados manualmente
